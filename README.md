## 📖 Topics Covered

### Core Services
- **Compute**: EC2, Lambda, ECS, Auto Scaling
- **Storage**: S3, EBS, EFS, Glacier
- **Database**: RDS, DynamoDB, ElastiCache
- **Networking**: VPC, ALB, Route 53, CloudFront
- **Messaging**: SNS, SQS, EventBridge
- **DevOps**: CloudFormation, CodePipeline, CloudWatch
- **Security**: IAM, KMS, Secrets Manager

## 🚀 Quick Start

1. **Prerequisites**:
   - AWS Free Tier Account (https://aws.amazon.com/free/)
   - AWS CLI installed locally
   - Basic understanding of Cloud Computing

2. **Setup**:
   ```bash
   git clone https://github.com/savlaj79/aws-learning.git
   cd aws-learning
   ```

3. **Start Learning**:
   - Begin with `01-AWS-Fundamentals/01-Core-Concepts.md`
   - Follow the sequential order
   - Complete each section before moving forward

## ⏱️ Time Estimates

- Fundamentals: 6-8 hours

- Interview Prep: 4-6 hours
- **Total**: ~20-30 hours
